# Profile translation built for LinkedIn structure

This page exists for users who are not trying to translate a casual sentence. They usually need to rewrite a complete LinkedIn profile that includes a headline, an About section, and multiple experience blocks.

## What this tool should help with

- Convert raw profile text into professional LinkedIn language.
- Preserve the structure of headline, summary, and role descriptions.
- Improve readability for recruiters who search in English.

## Recommended page experience

The interface should stay narrower than the homepage. Instead of a generic box, this page should feel like a guided profile workflow:

- Headline input
- About input
- Experience input
- A single output area that keeps the profile consistent

## SEO intent behind this page

Users landing here already know the job to be done. They are likely searching for phrases such as "translate LinkedIn profile to English" or "LinkedIn profile translator". The copy should therefore focus on clarity, profile sections, and bilingual career positioning.

## Conversion role

This page should still route users back to the main translator experience after search intent is satisfied. The content is explanatory, but the final action remains using the product.
